# Bento Loot Sell Repair

Automates merchant interactions and improves looting speed for streamlined gameplay.

## Features

- **Auto Sell Junk** - Automatically sells all junk items when you open a merchant window
- **Auto Repair** - Automatically repairs all equipment when visiting a merchant
- **Instant Loot** - Makes looting instant with no delay when auto-loot is enabled

## Installation

1. Download the addon
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI with `/reload`

## Usage

The addon works automatically once installed. Simply open a merchant window and your junk will be sold and gear repaired. Looting will be instant when auto-loot is enabled.
